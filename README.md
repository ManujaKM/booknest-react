# BookNest 📚

A modern book sharing platform built with React, Tailwind CSS, and Lucide icons.
Inspired by the Adopt a Book app (adoptabook.app).

## Features
- 🏠 Beautiful landing page with dark purple UI
- 📖 Book inventory management section
- 🔍 ISBN scanner feature showcase
- 👥 Community & social reading features
- 🔐 Multi-role login page (Customer, Admin, Delivery)
- 📱 Fully responsive — mobile, tablet, desktop

## Tech Stack
- React (Vite)
- Tailwind CSS
- Lucide React (icons)
- React Router DOM

## Getting Started

git clone https://github.com/yourusername/booknest-react.git
cd booknest-react
npm install
npm run dev
